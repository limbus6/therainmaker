# M&A Dealmaker Simulator

Jogo interactivo (single-page) para simular um processo de M&A desde o pitch ao closing.

## O que inclui

- Fases completas de um mandato de sell-side:
  - Pitch
  - Teaser / Information Memorandum
  - Contacto a compradores
  - Propostas não vinculativas
  - Due diligence
  - Negociação de SPA
  - Closing (waivers bancários e registos)
- Gestão de três tensões em simultâneo:
  - Relação com o cliente
  - Stress da equipa
  - Relação com compradores
- Indicadores de performance:
  - Valor estimado
  - Probabilidade de closing
  - Dias até fecho

## Compatibilidade GitHub Pages

A aplicação é estática e corre apenas com `index.html` (HTML+CSS+JavaScript), sendo compatível com publicação directa no GitHub Pages.

## Como usar localmente

Abrir `index.html` num browser.

## Nota de compatibilidade

Foram removidas APIs menos compatíveis (`prepend` e `Object.entries`) para evitar falhas em alguns ambientes de pré-visualização.
